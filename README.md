# P5.js Experiments

A collection of interactive visual experiments and creative coding demos built with [p5.js](https://p5js.org/).

![License](https://img.shields.io/badge/license-MIT-blue.svg)

| Experiment | Description |
|------------|-------------|
| **Particle Click** | Click anywhere to create a burst of colorful particles with physics-based animation |
| **Perlin Noise Flow Field** | Thousands of particles flow through a vector field generated by Perlin noise, creating organic swirling patterns |
| **Lorenz Attractor** | 3D visualization of chaos theory's famous "butterfly effect" strange attractor. Drag to rotate the view |

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/p5-experiments.git
cd p5-experiments

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will open in your browser at `http://localhost:5173`.

### Build for Production

```bash
npm run build
```

The production build will be output to the `dist/` directory.

## ğŸ¨ Adding a New Experiment

1. Create a new file in `src/experiments/` (e.g., `my-experiment.js`):

```javascript
import p5 from 'p5';

const sketch = (p) => {
    p.setup = () => {
        const canvas = p.createCanvas(p.windowWidth, p.windowHeight - 60);
        canvas.parent('canvas-container');
    };

    p.draw = () => {
        p.background(0);
        // Your drawing code here
    };

    p.windowResized = () => {
        p.resizeCanvas(p.windowWidth, p.windowHeight - 60);
    };
};

export default new p5(sketch);
```

2. Register the experiment in `src/js/main.js`:

```javascript
const experiments = [
    // ... existing experiments
    {
        id: 'my-experiment',
        title: 'My Experiment',
        description: 'Description of your experiment.',
        icon: 'ğŸ¨',
        module: () => import('../experiments/my-experiment.js')
    }
];
```

3. The experiment will automatically appear on the home page!
